== Description ==
[wp-ano-atual] ou [wp-ano-atual inicio = "© 2015"]
O plugin ideal para adicionar rapidamente o ano atual no rodapé do seu site via shortcode, muito simples. Apenas ative e coloque o shortcode onde quiser.
Exemplo de uso: [wp-ano-atual] ou [wp-ano-atual inicio = "© 2015"].
Resultado: 2015-2019

The ideal plugin to quickly add the current Year in the footer of your site via shortcode, very simple. Just activate and put the shortcode where you want.
Example usage: [wp-ano-atual] or [wp-ano-atual inicio = "© 2015"].
Result: 2015-2019
